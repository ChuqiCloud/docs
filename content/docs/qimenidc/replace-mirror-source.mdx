title: 更换PVE镜像源
description: Proxmox VE 设置国内源教程
---
<Callout>
  可以直接使用我们的一键脚本
</Callout>

```
wget -qO pve_change_repo.sh https://mirror.chuqiyun.com/install/pve_change_repo.sh && chmod +x pve_change_repo.sh && ./pve_change_repo.sh
```

```
# 换源（默认中科大）
sudo ./pve_change_repo.sh

# 换清华源
sudo ./pve_change_repo.sh tsinghua

# 一键还原
sudo ./pve_change_repo.sh restore
```
