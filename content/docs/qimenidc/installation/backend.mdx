## 服务器环境要求
1. Nginx
2. MySQL 5.7+（不能低于5.7）
3. Jdk 17+ （需自行安装）

## 部署环境
宝塔面板中点击 网站 - Java 项目 - 点击安装 - 选择安装版本 17
<img 
  src="https://img.xwyue.com/i/2025/07/20/687c7a4853d9b.png" 
  alt="安装Jdk"
  className="rounded-lg border border-fd-border"
/>


## 基础数据安装
上传`核心程序Jar包`以及`配置包`并解压

<Callout>
  后端程序运行目录可以自行选择创建
</Callout>

注意最新程序包可在：QimenIDC - 后端最新版下载及更新说明 查看下载

## 创建数据库
导入SQL目录下的`script.sql`文件

如上图上传后点击导入即可

修改基础配置文件：

配置文件在config目录下

主配置文件：application.yml

<Callout>
  profiles 的内容与 config 目录下的文件名对应，例如：profiles 为 prod 时，会加载 application-prod.yml 文件。
</Callout>

```
server:
  port: 8080

config:
  admin_path: admin
  secret: 123456
  frontend_DomainName: "http://localhost:3003"
  os_url: https://gitee.com/chuqicloud/soft-ware/raw/master/Cloud/os.json

MySQL:
  host: localhost
  port: 3306
  username: root
  password: 123456
  database: pve
```

将刚才添加的数据库配置信息复制到对应的位置然后保存即可

## 启动后端程序
点击添加Java项目：

项目jar路径选择 核心Jar包，端口可自定义修改（建议与config中的一致，该优先权>配置文件）：

点击提交

点开刚才创建的项目，可在项目日志中查看运行情况：
注意记录 管理员账号和密码

<Callout type="warning">
  注意该密码只出现一次，请直接记录下来保存后续使用，后续可以自行在面板上更改密码，注意只会显示一次
</Callout>
<Callout type="warning">
  注意项目端口为`/config`目录下配置文件中配置的端口，默认为`8080`端口，项目执行命令中`-Xmx4096M `代表最高允许使用内存， `-Xms256M` 为启动最低分配内存量，请根据自己服务器实际情况分配，`开机启动` 可自行开启
</Callout>
