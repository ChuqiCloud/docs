## Docker安装
### 服务器环境要求
1. Nginx
2. Docker

### 安装Docker
1. 直接使用宝塔Docker一键安装
2. 脚本一键安装
```
sudo wget -qO- https://get.docker.com/ | bash
```

### 一句话安装
<Callout>
  注意更换主控地址
</Callout>

```
wget -qO qimenidc_client.sh https://mirror.chuqiyun.com/install/qimenidc_client.sh && chmod +x qimenidc_client.sh && ./qimenidc_client.sh
```

```
# 安装
./qimenidc_client.sh install

# 更新
./qimenidc_client.sh update

# 卸载
./qimenidc_client.sh uninstall
```

<Callout>
  非交换一键脚本
</Callout>
```
export API_BASEURL=http://your-api.com && wget -qO- https://mirror.chuqiyun.com/install/qimenidc_client.sh | bash -s install
```

配置好您的域名即可使用了
